# An-lisis-y-claisifcaci-n-de-acciones-captadas-por-sensores-biom-tricos-
Análisis y claisifcación de acciones captadas por sensores biométricos (PYTHON)

Para ejecutar simplemente abrir el notebook de jupyter con JupyterNotebook e ir ejecutando celda a celda.

Las funciones que se encuentran en este notebook se pueden encontrar el los ficheros .py que se encuentran también en el repositorio.
El archivo MainDatos.py es un fichero de prueba de las diferentes funciones. Si se quiere ejecutar basta con escribir python3 MainDatos.py en la consola de comandos.
